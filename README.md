# 🌐 Proyecto: Sitio Web de Turismo - UNJu

## 📖 Descripción
Este proyecto corresponde a la práctica de **Desarrollo de un sitio web con HTML5 y CSS3** de la Facultad de Ingeniería, Universidad Nacional de Jujuy.  
El objetivo es implementar un sitio web semántico, responsivo y accesible, aplicando técnicas avanzadas de CSS y animaciones.

---

## 🎯 Objetivos
- Aplicar etiquetas semánticas de **HTML5**.
- Diseñar con **CSS3** utilizando el modelo de cajas, Flexbox y Grid.
- Implementar animaciones, transiciones y micro-interacciones.
- Garantizar accesibilidad y compatibilidad en distintos dispositivos.
- Publicar el sitio en línea mediante **GitHub Pages** o **Netlify**.

---

## 🛠️ Requisitos Técnicos
- **Editor recomendado:** Visual Studio Code  
  - Extensiones:  
    - *HTML CSS Support*  
    - *Live Server*  

- **Estructura del proyecto:**  
  - HTML5 para contenido (header, nav, footer, section, aside, article).  
  - CSS externo para estilos.  
  - Organización clara y modular.

---

## 📂 Estructura del Sitio Web

### 1. Página Principal (Home)
- Header con mega-menú desplegable (Destinos, Agencias, Precios).
- Hero con video de fondo y overlay animado.
- Destinos destacados en tarjetas con efectos hover.
- Contador animado con CSS Keyframes.
- Testimonios en carrusel CSS-only.
- Footer con:
  - Formulario de newsletter.
  - Enlaces a redes sociales.
  - Mapa interactivo embebido.

### 2. Página de Destinos
- Filtros por categoría usando `:checked` y combinadores CSS.
- Galería tipo *masonry* con CSS Grid.
- Tarjetas con zoom y overlay al hover.
- Tabla responsiva de precios.

### 3. Página de Agencias
- Tarjetas con efecto flip.
- Sistema de rating con estrellas animadas en CSS.

### 4. Formulario de Contacto
- Validación en tiempo real con `:valid` y `:invalid`.
- Spinner CSS al enviar.
- Diseño responsivo con Flexbox y Grid.
- Modal de confirmación al enviar.

### 5. Página de Precios
- Tabla comparativa con efecto hover.
- Tooltips explicativos.

### 6. Blog de Destinos
- Estilo tipo revista con CSS Grid.
- Sistema de tags y categorías (CSS-only).
- Comentarios estilizados con avatares en CSS.
- Efecto *Scroll Reveal*.

---

## 🎨 Técnicas Avanzadas de CSS
- Variables CSS (Custom Properties).
- Clip-path & Masking para imágenes.
- Scroll Snap para secciones.
- Dark Mode con CSS Variables.
- Animaciones con Keyframes y transiciones SVG.
- Micro-interacciones en botones y enlaces.

---

## ♿ Accesibilidad
- Contraste AA/AAA.
- Estados de focus visibles.
- Uso de ARIA labels.

## 📦 Entregables
1. Código HTML y CSS bien estructurado.
